# swiftbar_clocks

A CLI that displays different times in a format suitable for [SwiftBar][sb].
Most of the app was vibe-coded with Claude.

[sb]: https://github.com/swiftbar/SwiftBar

Exemplary config:

```yaml
cities:
  - name: Gdansk
    timezone: Europe/Warsaw
  - name: UTC
    timezone: Europe/London
  - name: Seattle
    timezone: US/Pacific
```
